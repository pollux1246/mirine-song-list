# mirine-song-list

[汐鳴みりねさん](https://www.youtube.com/@Mirine_Shionari)（ななしいんく所属）の歌ってみた、Shorts、歌枠での歌唱記録をまとめた非公式サイトです。

GitHub Pages：  
https://pollux1246.github.io/mirine-song-list/

## ファイル構成

- `index.html`：ページ本体
- `style.css`：デザインとレイアウト
- `script.js`：CSVの読み込みと表示処理
- `mirine_list.csv`：歌唱データ
  - `曲リンク`列は、管理用スプレッドシート上でリンクを表示するための列です。サイトでは使用していません。
